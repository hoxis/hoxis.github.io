k8s集群容器内部无法和集群外的Node通信

问题

k8s 集群下容器内部，只能和集群中的三台 Node 通信，无法和集群同 vlan 的网段地址其他 Node 通信，如 mysql 数据库服务器地址。

在容器外部的机器上是可以正常访问 MySQL 服务器的，但是在容器内部却不行：

解决过程

这个 k8s 集群使用 calico 网络方案。

发现 IPv6 的 NAT 没有打开。

calico 开启 natout 开关后，可以正常通信了！
